Installation

Server Requirements

  • PHP >= 5.6.4 (including PHP 7)
  • OpenSSL PHP Extension
  • PDO PHP Extension
  • Mbstring PHP Extension
  • Tokenizer PHP Extension
  • XML PHP Extension

Installation Process

  • Please have the elegant_1.9 file and upload it to the hosting server, Once upload extract it to your desire places
  • Create database and import elegant_1.9/public/db/database.sql
  • Open extracted directory and look at the root that you have a file called .env, change value of "DB_DATABASE,DB_USERNAME,DB_PASSWORD"
  • That's it!
