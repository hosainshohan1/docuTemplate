Installation

Server Requirements

  • PHP >= 5.6.4 (including PHP 7)
  • OpenSSL PHP Extension
  • PDO PHP Extension
  • Mbstring PHP Extension
  • Tokenizer PHP Extension
  • XML PHP Extension

Installation Process

  • Please upload the VenManager-V1.0.1.zip file to the hosting server
  • Once uploaded extract it to your desire place
  • Then create database and import public/db/database.sql
  • Open extracted directory and look at the root there you have a file called .env
  • If you didn't find the .env file then follow below insturctions
    • Press Settings File Manager
    • Tick Show Hidden Files (dotfiles)
    • You will see your .env file
  • Open the .env file and change value of "DB_DATABASE, DB_USERNAME, DB_PASSWORD"
  • That's it!
